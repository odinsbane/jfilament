Buttons

  • Previous Time: Changes from the current time to the previous time. This can be an expensive operation since it requires new rendering.
  • Next Time: Moves to the next time step
  • Lower Slice: Lowers slice in 2D view and 3D sections view.
  • Raise Slice: Raises slice
  • Add Snake: Create a new snake.
  • Delete Snake: Delete the selected snake
  • Deform Snake: Deforms the current snake until the number of iterations has been reached or the snake becomes too short.
  • Clear Current: Removes the current snake from this screen only.
  • Track Snake: Copies the current snake to the next time frame and deforms the snake.
  • Track Backwards: Copies the current snake to the previous time frame and deforms the snake.
  • Clear Screen: Removes all of the snakes in the current time frame.
  • Delete End Fix: Trims the ends of a snake. Click where you want to trim, then click near the end of the snake to be removed.
  • Delete Middle Fix: Click twice on the snake to remove a segment between two selected points (2 left clicks). A third left click places a new point. A third right click removes segment without placing a new point.
  • Stretch Fix: Extends the closest end of the snake to the next click location.
  • Zoom In: Zoom in the original 2D image by left clicking then right clicking, on the 2D image.
  • Zoom Out: Zoom out to the original 2D image.
  • +MAX: Increases maximum brightness of 3D display.
  • -MAX: Decreases maximum brightness of 3D display.
  • +MIN: Increases minimum brightness of 3D display.
  • -MIN: Decreases minimum brightness of 3D display.

Parameters

  • Alpha: Alpha determines the elasticity of the filament
  • Beta: Beta cause a filament to straighten, can cause filament to grow.
  • Gamma: Gamma restricts the distance the points move
  • Weight: Weight changes the level of influence the image has
  • Stretch Force: A factor to adjust the pulling force.
  • Deform Iterations: Number of iterations per deform snake click.(esc will stop)
  • Snake Point Separation:The distance between points when the contours are sampled
  • Image Smoothing:The width of the gaussian filter that is applied to the image.
  • Z Spacing:Distance between Z planes in x-y pixels. Changing this value will cause the 3D image to reload.
  • Foreground:Determines the pixel value where the snake will stretch while deforming.
  • Background:Complements the forground intensity. The snake will shrink when the snake is less than the mean of the Foreground and Background intensities.

Menus

  • Image: Load a different image.
  • Snakes: Load and save snakes.
  • Data: Save elongation data.
  • View: Change the display configuration, or launch the external VTK viewer.
  • help: Version and help information.

Keyboard Shortcuts

  • Arrow Keys move the planes in the 3D section view.
  • ",", and "." Change the current slice. Moves the X-Y plane up and down in 3D section view.
  • "Esc" Cancels deform snake operation.
  • "v" Cycles to next view configuration.
  • "f" Next time frame.
  • "b" Previous time frame.
